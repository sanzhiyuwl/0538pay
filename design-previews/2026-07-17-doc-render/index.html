方向说明

A Stripe 精致 refined · 推荐

气质
细边框+柔和阴影卡、endpoint 带渐变底、表格斑马纹、代码带窗口栏
字体
无衬线正文 + 等宽代码/参数名
记忆点
每种 block 都是一张精致小卡，专业 API 文档质感
适合
支付接口文档，开发者一眼觉得"靠谱、正规"

B Mintlify 清爽 airy

气质
大留白、圆角卡、彩色 callout 图标、胶囊方法徽章
字体
大字号无衬线，呼吸感强
记忆点
友好现代，像新一代 SaaS 文档站
适合
希望文档看起来年轻、亲和、不那么"硬核"

C 终端原生 terminal

气质
等宽字体主导、深色代码优先、# 和 $ 标记、极客感
字体
全等宽，命令行美学
记忆点
硬核开发者原生，像 README / man page
适合
纯技术受众，追求极客气质

D 杂志排版 editorial

气质
强对比、宽松行距、分隔线切段、下划线强调
字体
大字号无衬线 + 粗黑表头线
记忆点
阅读优先，像精心排版的技术长文
适合
长文档、教程型内容，重阅读体验
